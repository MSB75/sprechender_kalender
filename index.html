ℹ️ Über diese App

📱 Was macht diese App?

Sprechender Kalender ist eine barrierefreie Lern-App für Schülerinnen und Schüler mit dem Förderschwerpunkt geistige Entwicklung. Sie hilft dabei, den aktuellen Wochentag, das Datum, den Monat, das Jahr und die Jahreszeit zu benennen.

📵 Datenspeicherung

Diese App speichert keine persönlichen Daten. Es werden keine Daten auf externen Servern gespeichert oder übertragen. Die App funktioniert vollständig offline.

☁️ Hosting

Die App wird über GitHub Pages bereitgestellt. Beim Laden der Seite werden technische Zugriffsdaten (z. B. IP-Adresse, Zeitstempel) von GitHub Inc. verarbeitet. Dies ist für den Betrieb der Seite technisch notwendig. Weitere Informationen: GitHub Datenschutzerklärung.

🗓️ Jahreszeiten

Die Jahreszeiten richten sich nach dem kalendarischen Beginn:
Frühling: 20./21. März  ·  Sommer: 20./21. Juni
Herbst: 22./23. September  ·  Winter: 21./22. Dezember
